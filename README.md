# ACC102 Mini Assignment: GHG Emissions and GDP Analysis

## About This Project
This is a basic Python data analysis project for my ACC102 course.
I analyzed greenhouse gas (GHG) emissions from major economies and their relationship with GDP.

## Data Source
- GHG emissions data (2020–2023): World Bank Open Data
- GDP data (2023): Our World in Data
All data is real and publicly available.

## What I Did
1. Loaded and previewed the dataset using pandas
2. Created a bar chart to compare 2023 GHG emissions across countries
3. Made a scatter plot to explore the relationship between GDP and GHG emissions
4. Calculated and compared emission intensity (GHG per unit GDP)

## Files in This Repository
- `main.ipynb`: Complete Python code with outputs and charts
- `Reflection Report.docx`: My learning reflection
- `README.md`: This project overview

## Purpose
To practice basic Python data analysis and understand the link between economic activity and environmental impact.
